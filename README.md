# 🎮 Rock Paper Scissors

A modern and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript for **HackWeek 2026**.

---

## ✨ Features

- 🎮 Player vs Computer
- 🤖 Random computer moves
- 🏆 First to 5 wins
- 📊 Live scoreboard
- 🎉 Winner popup
- 🎊 Confetti celebration
- 💎 Glassmorphism UI
- 📱 Responsive design
- 🔄 Reset game

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
